## [1.0.0] - 2026-02-11

### 新增 (Added)
- **EsDialog** - 增强型对话框组件
  - 支持拖拽移动
  - 支持全屏切换
  - 支持自定义头部和底部
  - 支持嵌套对话框
  - 提供 `useDialog` 组合式 API

- **EsForm** - 增强型表单组件
  - JSON 配置式表单
  - 支持多种表单控件
  - 支持表单验证
  - 支持动态表单项

- **EsTable** - 增强型表格组件
  - 支持列宽拖拽调整
  - 支持自定义列配置
  - 支持分页
  - 支持自定义操作按钮

- **SvgIcon** - SVG 图标组件
  - 支持自定义图标类名
  - 支持动态图标加载

## 版本说明

### 版本类型
- **主版本 (Major)**：不兼容的 API 修改
- **次版本 (Minor)**：向下兼容的功能性新增
- **修订版本 (Patch)**：向下兼容的问题修正

### 变更类型
- **新增 (Added)**：新增功能
- **变更 (Changed)**：现有功能的变更
- **弃用 (Deprecated)**：即将移除的功能
- **移除 (Removed)**：已移除的功能
- **修复 (Fixed)**：bug 修复
- **安全 (Security)**：安全相关的修复
